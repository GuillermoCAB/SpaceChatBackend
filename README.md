# bunnies_backend
Backend do projeto BunniesHole

## Available Scripts

### `yarn start`
"node src/server.js"

### `yarn dev`
"nodemon src/server.js"

### `yarn dev:debug`
"nodemon --inspect src/server.js"
Configurado a opção de debug com -inspect, sendo assim não precisa parar o serviço para entrar em modo de debug
